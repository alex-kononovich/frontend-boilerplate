Develop with SASS, Compass, Slim and CoffeeScript
=================================================

Getting started
---------------

Clone the repo

```
$ git clone git://github.com/flskif/frontend-boilerplate.git YOUR_APP
```

Navigate to your brand new app

```
$ cd YOUR_APP
```

Install required gems

```
$ bundle install
```

And start server

```
$ rackup
```
